# 🐾 Vivra

El diario digital de tu mascota. App mobile-first, PWA instalable, para registrar vacunas, visitas al veterinario, alimentación, viajes y aventuras de tu perrito.

## Stack

- **Astro 5** (SSR) + **React** (islands)
- **Tailwind CSS 4**
- **Supabase** (Auth + Postgres + Storage)
- **Vercel** (deploy automático)
- **PWA** (instalable en móvil)

## Setup rápido

```bash
npm install
cp .env.example .env
# Edita .env con tus datos de Supabase
npm run dev
```

Ver instrucciones completas en la sección de abajo.
